# tmux ペイン参照の手順

> 本体（~/.claude/CLAUDE.md）からポインタで参照。「tmux の〇ペインを見て」系の依頼を受けたときに読む。

- 「tmuxの〇ペインを見て」と言われたら、現在のtmuxセッションの該当ペインを `tmux capture-pane -t {session}:{window}.{pane} -p` で取得する
- ペイン番号だけ指定された場合（例: 「1ペイン」「2ペイン」）は、同セッションのウィンドウ1の該当ペイン（1.1, 1.2, 1.3...）を参照する
- セッション名が不明な場合は `tmux list-panes -a` で一覧を確認して特定する
